## Core Rules (All Modes)
- Hebrew to user | Code/paths/commits in English
- Read only files needed for current task — never scan entire codebase
- NEVER duplicate existing functionality — search and reuse
- NEVER delete working code without explicit instruction
- No hardcoded secrets — env vars only
- async/await always — never .then().catch()
- No junk: *.bak *.tmp dead code commented-out code
- git add . && git commit after every completed task (conventional commits)
- After every change: confirm nothing existing broke
